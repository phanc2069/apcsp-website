Linux Reference Page

Element Description Example
uname -a This command gives the specs of the device you are currently on uname -a --> Linux rpi11 4.19.75-v7+ #1270 SMP Tue Sep 24 18:45:11 BST 2019 armv7l GNU/Linux
Echo Types the words on to the channel sp1b@rpi11:~ $ echo hello world
date Gives the real time and date to the second sp1b@rpi11:~ $ date Tue 03 Dec 2019 12:07:14 PM PST
hostname Gives the name of the device sp1b@rpi11:~ $ hostname rpi11
arch Gives the processor's model sp1b@rpi11:~ $ arch armv7l
uptime Gives the amount of time and resources used by the users in the machine sp1b@rpi11:~ $ uptime 12:19:18 up 1 day, 4:59, 2 users, load average: 0.02, 0.01, 0.00
who Gives information about which user is on the device sp1b@rpi11:~ $ who sp1a pts/0 2019-12-03 11:53 (172.19.32.60) sp1b pts/1 2019-12-03 12:03 (172.19.32.52)
yes infinitely repeats the string yes yes yes...

w Gives the time of login and how long they went idle sp1b@rpi11:~ $ w 12:23:09 up 1 day, 5:03, 2 users, load average: 0.00, 0.00, 0.00 USER TTY FROM LOGIN@ IDLE JCPU PCPU WHAT sp1a pts/0 172.19.32.60 11:53 11:57 0.45s 0.34s -bash sp1b pts/1 172.19.32.52 12:03 1.00s 0.35s 0.03s w
